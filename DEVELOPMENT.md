# Development Setup

Get from zero to hot-reloading plugin changes in Obsidian in under 10 minutes.

The test vault lives at `test/` inside this directory. No symlinks needed —
`npm run dev` outputs `main.js` directly into `test/.obsidian/plugins/obsidian-tracker/`.

---

## Prerequisites

- [Node.js](https://nodejs.org/) v18+ (verify: `node -v`)
- [Obsidian](https://obsidian.md/) desktop app installed

---

## Step 1 — Install dependencies

Open a terminal in the plugin source directory
(`C:\Users\elias\OneDrive\Desktop\testvaulthome`) and run:

```bash
npm install
```

---

## Step 2 — Install Hot Reload in the test vault

Hot Reload watches for `main.js` changes and reloads the plugin without
restarting Obsidian.

1. Download the latest release from: https://github.com/pjeby/hot-reload/releases
   - Grab `hot-reload.zip` (or the individual files: `main.js`, `manifest.json`)
2. Extract/copy them into:
   `testvaulthome\test\.obsidian\plugins\hot-reload\`
   (create the `hot-reload` folder if it doesn't exist)
3. The folder should contain: `main.js` and `manifest.json`

---

## Step 3 — Run the dev build

```bash
npm run dev
```

This will:
- Compile `src/main.ts` → `test/.obsidian/plugins/obsidian-tracker/main.js`
- Copy `manifest.json` and `styles.css` into the same folder
- Create a `.hotreload` marker file (triggers Hot Reload to watch this plugin)
- Watch for source file changes and rebuild in ~100ms

Leave this terminal running.

---

## Step 4 — Open the test vault in Obsidian

1. Open Obsidian.
2. Click **Open another vault → Open folder as vault**.
3. Select `C:\Users\elias\OneDrive\Desktop\testvaulthome\test`.
4. If prompted about safe mode, click **Turn on community plugins**.
5. Go to **Settings → Community plugins → Installed plugins**.
6. Enable **Hot Reload**.
7. Enable **Study & Habits Tracker**.
8. Open the developer console: **Ctrl+Shift+I** → Console tab.
9. You should see: `Study & Habits Tracker: loaded`

---

## Step 5 — Verify hot reload works

1. Open `src/main.ts`.
2. Change the log message to anything, e.g. `"Study & Habits Tracker: reloaded!"`.
3. Save the file.
4. Within ~1 second, Obsidian's console should show the new message.

**You are now in the hot-reload loop.** Edit → save → see changes instantly.

| Command | What it does |

## Troubleshooting

**Plugin doesn't appear in Obsidian:**
- Confirm `npm run dev` ran successfully and `test/.obsidian/plugins/obsidian-tracker/main.js` exists
- Confirm `manifest.json` exists in the same folder
- Restart Obsidian and check Settings → Community plugins

**Hot Reload isn't picking up changes:**
- Confirm `.hotreload` file exists in `test/.obsidian/plugins/obsidian-tracker/`
  (it's created automatically by `npm run dev`)
- Confirm the Hot Reload plugin is enabled in Settings
- Check the Obsidian console (Ctrl+Shift+I) for errors

**Build / TypeScript errors:**
- Run `npm install` to make sure all dependencies are installed
- TypeScript errors appear in the terminal running `npm run dev`

**`npm run dev` exits immediately:**
- You may have a syntax error in `src/main.ts` — check the terminal output
